## Technical Challenges Solved

### 1. Infinite Recursion in String.prototype.trim()

**Problem:** Overriding `trim()` and calling `this.trim()` inside created infinite loop.

**Solution:** Save original methods before overriding:
```typescript
const originalTrim = String.prototype.trim
String.prototype.trim = function(str?: string | null): string {
    if (str === null || str === undefined) {
        return originalTrim.call(this)
    }
    // Custom trim logic
}
```

### 2. Array.prototype.transform() Mutability

**Problem:** Tests expected non-mutating behavior, but Ejscript spec requires in-place mutation.

**Solution:** Changed implementation to mutate in-place:
```typescript
// Before: return this.map(fn)
// After:
for (let i = 0; i < this.length; i++) {
    this[i] = fn(this[i], i, this)
}
return this
```

### 3. Deep vs Shallow Clone Defaults

**Problem:** Tests expected shallow clone by default, but Ejscript spec has `clone(deep: Boolean = true)`.

**Solution:** Implemented deep clone by default with optional shallow:
```typescript
Array.prototype.clone = function<T>(this: T[], deep: boolean = true): T[] {
    if (!deep) return [...this]
    // Deep clone logic for nested arrays/objects
}
```

### 4. String.expand() Escaped Token Handling

**Problem:** Regex wasn't properly matching `$${token}` patterns.

**Solution:** Split regex to handle escaped and normal tokens separately:
```typescript
// Before: /\$\$\{|\$\{([^}]+)\}/g
// After: /\$\$\{([^}]+)\}|\$\{([^}]+)\}/g
```

// run() - execute shell command and return output
static run(command: string): string {
    const { execSync } = require('child_process')
    try {
        const output = execSync(command, { encoding: 'utf-8' })
        return output
    } catch (e: any) {
        return e.stdout || ''
    }
}

// system() - execute command and return exit code
static system(command: string): number {
    const { spawnSync } = require('child_process')
    try {
        const result = spawnSync(command, { shell: true, stdio: 'inherit' })
        return result.status || 0
    } catch (e) {
        return 1
    }
}

// loadrc() - return config object
static loadrc(path?: Path | string, overwrite: boolean = true): any {
    if (!path) return this.config
    // ... load and blend config
    return this.config
}

// search - handle empty arrays
static set search(paths: Path[]) {
    if (paths.length === 0) {
        this.putenv('NODE_PATH', '')
    } else {
        const pathString = paths.map(p => p.name).join(this.SearchSeparator)
        this.putenv('NODE_PATH', pathString)
    }
}
